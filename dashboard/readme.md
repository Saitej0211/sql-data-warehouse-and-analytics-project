# Tableau Dashboard Development Workflow

## 1. Analyze Requirements
- Collect requirements

## 2. Choose the Right Charts
- Draw mockups  
- Choose appropriate chart types  
- Select color schemes

## 3. Build the Data Source
- Connect data  
- Create data model  
- Rename fields/tables  
- Check data types  
- Understand the data

## 4. Build Charts
- Create calculated fields & test  
- Build charts  
- Format visual elements  
    - Remove lines & grids  
    - Clean up axis & headers  
    - Apply consistent coloring  
    - Add informative tooltips

## 5. Build the Dashboard
- Draw mockups for containers  
- Build container structure  
- Combine all charts together  
- Format layout and design  
    - Distribute content evenly  
    - Adjust colors, sizes, and alignment  
    - Fit to entire view  
    - Add legends  
    - Add inner/outer padding for spacing  
- Add filters & dynamic interactivity  
- Add icons for better UX
